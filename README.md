# Sistema-Minimarket

Sistema de Gestión de Minimarket desarrollado como aplicación web utilizando **Java Web (JSP y Servlets)**, **MySQL** y **Apache Tomcat**.

## Funcionalidades principales
- Gestión de productos y categorías.
- Control de inventario y lotes.
- Gestión de proveedores.
- Registro de ventas.
- Gestión de usuarios y roles.
- Reportes.
- Auditoría de acciones del sistema.

## Tecnologías utilizadas
- Java
- JSP y Servlets
- MySQL
- Apache Tomcat
- Eclipse IDE
- HTML, CSS, JavaScript y Bootstrap
